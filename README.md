# Fast-Fashion-Impact-on-Environment

# Project Overview
This interactive Power BI dashboard visualizes and analyzes the environmental toll of the fast fashion industry. The goal of this project is to highlight critical metrics regarding water consumption, carbon emissions, textile waste, and consumer behavior to drive awareness and data-driven insights into sustainable fashion practices.

# Key Features & Insights
Carbon & Water Footprint Analysis: Tracks the total greenhouse gas emissions (CO2e) and liters of water required to produce common garments (e.g., denim, cotton t-shirts).

Waste Lifecycle Tracking: Visualizes the journey of textiles from production to landfills, highlighting recycling vs. waste percentages.

Consumer Habits & Trends: Analyzes garment lifespan, average garments purchased per year, and the growth rate of the fast fashion market vs. sustainable alternatives.

Country/Regional Impacts: A geographical breakdown of where textile production impacts environments the most versus where major waste is generated.

# Tech Stack & Skills Showcased
Power BI Desktop: Report authoring, dashboard orchestration and UX/UI design.
Power Query (M Script): Data extraction, transformation and cleansing (ETL) to normalize environmental metrics.
DAX (Data Analysis Expressions): Created custom metrics, time-intelligence calculations and dynamic parameters for what-if sustainability scenarios.
Data Modeling: Developed a clean Star Schema utilizing centralized fact tables for environmental metrics connected to dimension tables for geography, garment types and time.

Data Sources
The data utilized in this report was aggregated from publicly available environmental studies and reports (e.g., Pulse of the Fashion Industry, Ellen MacArthur Foundation, UN Environment Programme).
